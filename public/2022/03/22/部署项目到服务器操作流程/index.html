下载 File Zilla Client

下载并安装 file zilla client（下载地址：https://filezilla-project.org）。

连接服务器

如图一所示，在 file zilla 中分别输入主机名、用户名、密码、端口，然后点击快速连接接入服务器，出现的弹窗全部点确定。
