引言

Tailwind CSS 是一个功能类优先的 CSS 框架，通过预定义的类名来实现某些特定功能的 css 样式，可以帮助开发者快速地自定义地实现预想的布局和样式设计，大大地提高界面的开发效率。本文通过一个实例展示 tailwind 使用流程。

后记

有了 tailwind，写样式就灵活、快捷了许多，几乎不用再费劲地写很多重复、很难维护的 css 了。但是 tailwind 有相当数量的预定义类名，对于熟悉 css 的开发者，掌握到它们的命名规则可以很容易记住，但是为了开发的便利，对于 vscode 用户，可以安装插件 Tailwind CSS IntelliSense。
